# android-map-keyword
**카카오 테크 캠퍼스 STEP2 Android 2주차 미션 - 검색어 저장**
## 기능 요구사항
### 1단계 기능
- 검색어 입력 및 검색 결과를 표시할 기본 레이아웃을 구현한다.
- 검색에 사용될 데이터를 로컬 데이터베이스에 생성한다.

## 프로그래밍 요구사항
- 검색 데이터 저장은 `SQLite`를 사용한다.
- 가능한 MVVM 아키텍처 패턴을 적용하도록 한다.
- 코드 컨벤션을 준수하여 프로그래밍 한다.

## 구현할 기능 목록
### 1단계 기능
- [x] 검색어 입력 레이아웃
- [x] 검색 결과 레이아웃
- [ ] 검색 데이터 저장할 데이터베이스
- [ ] MVVM 패턴 적용
